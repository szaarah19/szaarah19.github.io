# Personal Portfolio Website

## Overview
This is my personal portfolio website showcasing my software and game development projects, technical skills, and academic background as a Computer Science student.

The website was built using HTML and CSS to create a clean, responsive, and professional layout.

## Features
- Structured multi-section layout (About, Projects, Skills, Contact)
- Organized project cards for clear presentation
- Clean and minimal UI design
- Responsive styling for readability

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Poppins)

## Projects Featured
- Pet Adoption Center Application (Python + Tkinter GUI)
- Games by Sage (Roblox Game Development using Lua, Blender, Ibis Paint X)

## Purpose
The goal of this project was to create a professional online presence to present my technical skills and development projects to recruiters and internship opportunities.
